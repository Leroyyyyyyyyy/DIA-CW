# Agent Memory

## 1. 时区与本地情绪偏差

- 所在时区赔率往往受本地散户情感驱动。
- 典型表现是本地用户会过分看好本地选手、本地战队或本地叙事。
- 对这类市场要警惕情绪溢价，不要把本地热度直接当成真实胜率。

## 2. Multi-Agent / Self-Evolve

- 顶级传统量化背景的人一旦开始认真尝试 multi-agent 和 self-evolve，会非常快意识到其潜力。
- 这说明后续 agent 设计不能只停留在单策略、单规则、静态逻辑。
- 需要持续关注：
  - multi-agent 协作
  - self-evolve / self-improvement
  - 自动策略迭代
- 普通参与者往往低估这类系统对交易和研究流程的改变速度。

## 3. 高低胜率仓位的处理纪律

- 胜率低的标的，如果胜率上涨了，应该优先考虑卖出兑现。
- 胜率高的标的，通常是因为有足够强的基本面或信息理由支撑，不要轻易过早卖掉。
- 如果胜率已经到 90% 左右，通常翻盘空间很小，可以倾向于继续拿住。

## 4. 市场筛选原则

- 优先找流动性高、奖励高的世界赛市场。
- 尽量避免小众、低流动性、潜在假赛风险高的比赛。
- 市场选择本身就是策略的一部分，不要把所有比赛都当成同质量机会。

## 5. 低胜率市场的风险控制

- 尽量不去预测胜率低的标的。
- 如果确实参与了低胜率标的，必须有更严格的止损纪律。
- 一个明确规则是：如果价格跌到 10% 左右，要及时止损。

## 6. 领先方的翻盘概率判断

- 当领先方已经接近胜利时，落后方追分后完成翻盘的概率通常很低。
- 这个判断在双方原本就存在明显实力差距时更成立。
- 因此在强队领先、比赛进入后段时，不要轻易因为短时追分而高估落后方翻盘概率。
